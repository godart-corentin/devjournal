# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues,
pull requests, or discussions.

Instead, contact the maintainer privately through the contact methods listed on
the maintainer's GitHub profile: <https://github.com/godart-corentin>.

When possible, include:

- A description of the issue and its potential impact
- Steps to reproduce the problem
- The affected version, platform, and configuration
- Any suggested mitigation or fix

We will review reports as quickly as possible, aim to confirm receipt promptly,
and coordinate on disclosure once a fix or mitigation is available.
